# Sales SQL Analysis - Superstore Dataset

This project demonstrates how to use SQL and Python together for real-world sales data analysis.

## Dataset
SampleSuperstore.csv (retail sales dataset)

## What this project does
- Loads CSV data into SQLite database
- Runs SQL queries to analyze sales and profit
- Identifies top products and best performing regions
- Demonstrates practical SQL + Python workflow

## Technologies
Python, Pandas, SQLite, SQL
